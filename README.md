# shopify

## Mini E-Commerce App

A simple front-end e-commerce application built using HTML, CSS, and JavaScript.

---

## Features

- Product listing from JSON
- Search, filter, and sort
- Add to cart with quantity controls
- Wishlist functionality
- Product detail modal
- Basic checkout flow 

---

## Tech Stack

- HTML
- CSS
- JavaScript
